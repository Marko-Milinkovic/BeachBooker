1. Uvod

1.1 Namena

Baza podataka za BeachBooker centralizuje podatke o korisnicima, beach barovima, ležaljkama, rezervacijama i pratećim entitetima. Backend je Django; klijent koristi HTML i AJAX za dinamičke mape i rezervacije. Dokument sadrži ER prikaz, IE dijagram, relacione šeme i katalog tabela — osnova za implementaciju i odbranu projekta.

DBMS: MySQL 8+ · Šema: beachbooker

1.2 Ciljne grupe

Vođa tima, projektant baze, backend razvoj i ostali članovi tima — za usklađivanje imena polja, integraciju i održavanje modela.

1.3 Organizacija dokumenta

Poglavlje 2 — modeli; poglavlje 3 — tabele; prilog — SQL skripte u repozitorijumu (schema.sql).
